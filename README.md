# PrintQueue

A React web application for managing patron tickers, tracking plate statuses, and integrating with Supabase as the backend database.

---

## Features

- **View Orders** – See all patron orders in a clean grid layout.  
- **Edit Orders** – Update patron information, plates, billing status, and staff assignments.  
- **Track Plates** – Mark plates as “Printing” or “Done” with a checkbox toggle.  
- **Delete Orders** – Remove orders from the database.  
- **Real-Time Updates** – Changes are reflected immediately after editing or deleting orders.  
- **Responsive Design** – Works on desktop and mobile screens.  

---

## Tech Stack

- **Frontend:** React, React Router, CSS  
- **Backend:** Supabase (PostgreSQL)  
- **Components:**  
  - Navbar  
  - Modal
